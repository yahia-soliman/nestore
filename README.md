## Description

Prototype for a yet-small but extensible e-commerce application server build with [Nest](https://github.com/nestjs/nest) framework and [TypeScript](https://github.com/microsoft/TypeScript).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
